Desafio: Batalha Naval (Nível Novato)
Este projeto consiste na implementação inicial de um tabuleiro de Batalha Naval, desenvolvido como parte do desafio de lógica de programação em C.

 Descrição do Projeto
O programa cria um tabuleiro de 10x10 e posiciona estrategicamente dois navios de tamanho 3: um na horizontal e outro na vertical. O código foi estruturado para garantir a legibilidade, a organização e o cumprimento de todos os requisitos funcionais.

 Requisitos Implementados
Tabuleiro: Matriz bidimensional (10x10) inicializada com o valor 0 (Água).

Posicionamento: * Navio Horizontal (Tamanho 3) representado pelo valor 3.

Navio Vertical (Tamanho 3) representado pelo valor 3.

Validação: Implementação de lógica para garantir que os navios não ultrapassem os limites do tabuleiro e não se sobreponham.

Exibição: Impressão formatada do tabuleiro no console para fácil visualização.

Tecnologias e Conceitos
Linguagem: C

Matrizes: Manipulação de arrays bidimensionais.

Loops: Uso de laços aninhados (for) para processamento de dados.

Documentação: Código comentado seguindo boas práticas de desenvolvimento.

 Como Executar
Compile o arquivo:

Bash

gcc main.c -o batalha_naval
Execute o programa:

Bash

./batalha_naval
